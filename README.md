# SannePedersen

Personlig hjemmeside for Sanne Pedersen.

## Om Projektet

Dette er en enkel, responsiv hjemmeside bygget med HTML og CSS.

## Sådan Kommer Du I Gang

1. Clone dette repository
2. Åbn `index.html` i din browser

Du kan også køre projektet med en lokal webserver:

```bash
# Med Python 3
python -m http.server 8000

# Med Node.js (npx http-server)
npx http-server
```

Besøg derefter http://localhost:8000 i din browser.

## Struktur

- `index.html` - Hovedsiden
- `style.css` - Styling og layout
- `.gitignore` - Git ignore konfiguration
